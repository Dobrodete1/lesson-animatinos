# Animatins and keyframes

## git cheatsheet

```bash
# инициализация git
git init

# проверка наличия изменений
git status

# выбрать все изменения
git add -A
# или
git add .

# сохранить изменения
git commit -m "innit commit"

# добавить удаленное подключение ( репозиторий )
git remote add origin git@github.com:Dobrodete1/lesson-animatinos.git

# проверка подключений
git remote -v

#отправка изменений на удаленный репозиторий
git push origin master
```
